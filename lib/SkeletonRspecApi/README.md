# Httparty_rspec
Um projeto completo usando rspec, httparty e ruby para servir de exemplo

## 1-Passo
Depois entre na pasta

```
capyabra_rspec/tests
```

## 2-Passo
Agora rode o comando:

```
bundle install
```

## 3-Passo
Para rodar o projeto basta usar o comando:

```
bundle exec rspec
```
